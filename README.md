# **Social Spark - Backend**

The backend for **Social Spark** is built with **Node.js**, **Express.js**, and **MongoDB**. It provides API endpoints for event management, user authentication, and contributions tracking.

This backend handles:

- **User Authentication**: Secure login/registration using Firebase Authentication (email/password, Google login).
- **Event Management**: CRUD operations for creating, updating, deleting, and fetching events.
- **Contribution Tracking**: Allows users to contribute to events and track their contribution history.
- **Reporting**: Ability to generate contribution reports and manage event data.
- **Dynamic Routes**: API endpoints for various actions, including joining events, creating events, etc.
